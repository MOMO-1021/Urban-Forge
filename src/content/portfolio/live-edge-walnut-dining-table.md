## Project Overview

This custom dining table was commissioned for a large open-concept dining space overlooking Lake Washington. The goal was to fuse organic warmth with architectural strength.

## Fabrication Details

The base consists of massive 3/4" thick steel plates, faceted and welded into a pair of striking asymmetric pedestals. We incorporated subtle satin brass through-tenons that secure the wood slab to the base while allowing for seasonal wood movement.
